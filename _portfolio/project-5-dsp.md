Researched and implemented Disciplined Saddle Programming (DSP) frameworks to solve saddle point problems in convex optimization with real-world applications.

## Applications
- Resource allocation optimization
- Portfolio optimization
- Machine learning optimization problems

## Results
- Improved computational efficiency vs traditional methods
- Enhanced robustness in solving large-scale optimization problems

Python, MATLAB, CVXPY, SciPy, NumPy
